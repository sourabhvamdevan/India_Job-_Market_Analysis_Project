# India_Job-_Market_Analysis_Project

* **Objective**

  * Analyze the Indian Job Market dataset containing **20,000 job listings**.
  * Identify **job market trends** across different locations and industries.
  * Examine **salary distributions** based on experience, job type, and location.
  * Discover the **most in-demand skills** and hiring patterns.
  * Analyze **remote, onsite, and hybrid** work opportunities.
  * Provide actionable insights to help **job seekers, recruiters, and organizations** make data-driven decisions.

* **Approach & Tools Used**

  * **Excel**

    * Performed data cleaning and preprocessing.
    * Handled missing values and removed inconsistencies.
    * Standardized data formats for accurate analysis.
    * Conducted initial Exploratory Data Analysis (EDA) and generated summary statistics.

  * **SQL**

    * Queried and analyzed the cleaned dataset.
    * Identified top hiring companies and job locations.
    * Analyzed salary trends across experience levels.
    * Extracted insights on in-demand skills and job types.
    * Evaluated remote hiring opportunities and company-wise hiring patterns.

  * **Power BI**

    * Built an interactive dashboard for data visualization.
    * Created charts for salary trends, job types, hiring companies, and job locations.
    * Added interactive filters and slicers for dynamic exploration.
    * Enabled users to analyze insights based on experience, company, location, and employment type.
    * Presented key findings in a clear and user-friendly visual format.
